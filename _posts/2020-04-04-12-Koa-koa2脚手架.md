# `Koa`脚手架   最好还是自己搭建比较适合

  ##### 通过脚手架，可以快速生成一个`koa2`应用的骨架

1. 安装脚手架

   ```javascript
   npm install koa-generator -g //全局安装koa2 的脚手架
   ```

   

2. 利用`koa2`脚手架创建项目

   ```javascript
   koa xxx项目名称
   
   用vscode 打开项目
   
   //安装依赖
   npm install 
   
   
   //这样一个koa2搭建的项目的骨架就搭建好了
   ```
